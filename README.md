# LED Project
## Demo Video
https://youtu.be/jABRUeiTWB4
## Dependency 
C
## Setup
Hardware: Rasberry Pi 3 Model B V1.2  
Software: Raspbian 3.2.1
## Usage
LEDをRasberry Piの22番と39番のピンに接続  
myled.cをmakefileでコンパイル 
    sudo insmod myled.ko
